The biogas plants take biomass waste — frequently from agriculture — and use industrial bioreactors to turn it into methane and carbon dioxide. BioCirc captures the carbon dioxide and stores it in an underground reservoir offshore. The methane is then burned in a power plant. Four AI supply-chain attacks in 50 days exposed the release pipeline red teams aren't covering

요약

In a span of 50 days, four AI supply-chain attacks affected OpenAI, Anthropic, and Meta, highlighting a critical vulnerability in release pipelines, dependency hooks, CI runners, and packaging gates that current red team exercises fail to address. These incidents included three adversary-driven attacks and one self-inflicted packaging failure, none of which targeted the AI models themselves. One notable attack involved a self-propagating worm, "Mini Shai-Hulud," which published 84 malicious package versions across 42 npm packages by exploiting a misconfiguration in GitHub Actions and OIDC token extraction. Separately, OpenAI confirmed that two employee devices were compromised, leading to credential exfiltration from internal code repositories, prompting them to revoke macOS security certificates and enforce updates. These events underscore that model red teams are not adequately covering release pipelines, a significant architectural finding for AI vendors.
